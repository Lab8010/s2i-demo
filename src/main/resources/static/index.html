RHEL Certification Puzzle

RHEL 認定資格トラックの試験ブロックをドラッグし、レベル条件を満たしてランプを点灯させよう。

RHEL 試験ブロック

EX200 から各種スペシャリスト試験まで、適切なブロックをターゲットエリアに入れてください。

RHEL 認定ターゲット

ここに認定試験ブロックをすべて入れてください
EX200 などの試験ブロックをドラッグしてください

Level 2: EX200 + EX342

遊び方

  • 試験ブロックをドラッグしてターゲットエリアに追加します。
  • 必要な組み合わせを揃えると、対応する Level ランプが点灯します。
  • Level 3 以上は、EX200 と EX342 に加えてスペシャリスト試験を複数選択します。
